
# 🏥 zweitmeinung.ng Design System

Willkommen zum **Storybook** der Healthcare-Platform!

## 🎯 Was ist hier verfügbar?

### **Healthcare Design System**
- **Farbpalette**: Medizinisch optimierte Farben für Vertrauen und Beruhigung
- **Typography**: Roboto Condensed für professionelle Lesbarkeit  
- **Components**: WCAG 2.1 AA konforme UI-Komponenten

### **Accessibility First**
- **Touch-Targets**: Mindestens 56px für Healthcare-Kontext
- **Kontrast**: 4.5:1 Verhältnis für alle Texte
- **Screen Reader**: Vollständige ARIA-Unterstützung

## 🚀 Quick Start

1. **Browse Components** - Erkunden Sie die verfügbaren Healthcare-Components
2. **Test Accessibility** - Nutzen Sie den A11y Tab für Barrierefreiheits-Tests  
3. **Interactive Controls** - Testen Sie Components mit den Control-Panels
4. **Documentation** - Jede Component hat automatisch generierte Dokumentation
